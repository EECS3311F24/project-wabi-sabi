1.2. elaboration

1.2.1. general blueprint structure:

  • init.py: To make a blueprint.
  • routes.py: routing for blueprint
  • forms.py: forms for blueprint
  • models: data models for blueprint

1.2.2. blueprints:

/main: not auth stuff, has sub blueprints:

  • /pomodoro: timer
  • /tasks: task management
  • /friends: friends
  • /stats: stats about your studying

/auth: handles authentication

1.2.3. other directories:

/extensions: handles instances of plugins (db, login) /migrations: handles db migrations

1.2.4. other files:

seeds.py: adds sample data to the database config.py: configure the flask app run.py: runs the flask app requirements.txt: package management
